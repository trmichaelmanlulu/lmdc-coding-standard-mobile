2. Global styles should be declared first before the component styles

// dont
const styles = StyleSheet.create({
  // component styles
  
  //global styles
});

// do
const styles = StyleSheet.create({
  //global styles
	
  // component styles
});

3. Make you style name more specific as possible

We can follow this format when naming our styles

Format:
<adjectiveOfElement (optional)><NameOfElelement><PurposeOfElement>

// dont
cardStyle
cardBox

// do
mainCardContainer
roundedCardContainer
cardContainer

4. Initialize your styles before using it on the template. Order of style constant is important.

The order of style constant will follow the order of the style being used on the template.

// dont
<View style={styles.priceContainer}>
	// ...
</View>

// do
const {
	cardsMainContainer,
	cardsContainer,
	priceContainer,
	descriptionContainer
} = styles;

<View style={cardsMainContainer}>
    <View style={cardsContainer}>
        <View style={priceContainer}>
	        // ...
        </View>
        <View style={descriptionContainer}>
	        // ...
        </View>
    </View>
</View>

5. Separate your element styles based on their type and should follow this order

Always follow this order when styling your element. If an element has multiple style type, separate them with a comment

  • Positioning
    • ex. relative, absolute, top, bottom, left, right, z-index
  • Display
    • ex. flex, flex values (justify-content, align-items) , block, inline, none
  • Typography
    • ex. font-size, font-family, line-height
  • Colors
    • ex. colors, background-color
  • Width & Height
    • ex. width, height, min-width, min-height
  • Spacing
    • ex. padding, margin
  • Others
    • ex. border, border-raidus, overflow
// dont
    container: {
	    width: 140,
	    height: 140,
	    borderRadius: 140 / 2,
	    position: absolute,
	    left: 0,
	    top: 0,
	    alignSelf: 'center',
	    borderWidth: 8,
	    marginVertical: 4,
	    overflow: 'hidden',
	    justifyContent: 'flex-end',
	    borderColor: COLOR.BORDER_PRIMARY,
	    backgroundColor: 'white',
	    fontSize: 18,
	    fontFamily: 'Lato'
    }
    
// do
	container: {
		// position
		position: absolute,
	    left: 0,
	    top: 0,
	    // display
	    justifyContent: 'flex-end',
	    alignSelf: 'center',
	    // typography
	    fontFamily: 'Lato',
		fontSize: 18,
		// color
	    backgroundColor: 'white',
		borderColor: COLOR.BORDER_PRIMARY,
		// width and height
		width: 140,
	    height: 140,
	    // spacing
		marginVertical: 4,
		// others
		borderWidth: 8,
		overflow: 'hidden'
	}

6. Number values and unit on style

  • Number values should be divisible by 8 to make our application symmetrical.
    • Use 4 if 8 is not possible
  • Use % unit for width and heights. Use static value for the others.
