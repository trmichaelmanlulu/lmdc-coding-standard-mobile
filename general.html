LMDC Coding Standard (Mobile)

General

1. Use single quote instead of double quote in all javascript files

These apply to all template, styles and script files.

2. Proper spacing and indention

Make sure every element in your code has a space before and after.

// dont
const {itemOne, itemTwo}= object;

// do
const { itemOne, itemTwo } = object;

3. If object items are more than 3, separate them in new line

// dont
const { itemOne, itemTwo, itemThree, itemFour }= object;

// do
const {
    itemOne,
    itemTwo,
    itemThree,
    itemFour
} = object;
